Beer App 1.0 - Made with react 16.0.0 and Webpack 3.1.0

Created by Claudio Perizzolo

To run it:

1) Go to the project's folder in the command line (by default beer-app)

2) Type and execute "npm install" or "yarn install" to install the necessary dependencies

3) Type and execute "yarn run dev-server" or "npm run dev-server"

4) The application will be running by default at http://localhost:8080/
